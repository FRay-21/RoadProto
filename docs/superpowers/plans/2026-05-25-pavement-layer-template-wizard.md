# 路面结构层创建向导实施计划

> 本文件为历史实施计划的中文归档版。以后 `docs/superpowers/plans/` 下的计划文档统一使用中文编写；代码标识、命令名、路径和构建命令保持原文。

## 目标

在路面结构层模板创建流程前增加 WPF 创建向导，按路面类型和适应路段类型生成初始结构层参数。

## 主要任务

- 在 WPF 中新增创建向导窗口和预设工厂。
- 调整托管命令，使新建模板先显示向导，旧实体编辑仍直接打开参数窗口。
- 扩展 Bridge DTO，传递向导预设、内外侧厚度、加宽、坡度和填充参数。
- 更新核心测试、托管 Bridge 测试、业务文档和版本记录。

## 验证

- 托管测试覆盖向导预设、默认预览宽度、XML 往返和窗口控件。
- AutoCAD 手工验证创建向导、参数窗口、实体生成和编辑路径。
