# 平曲线参数编辑

## 基本信息

- 功能名称：平曲线参数编辑
- 所属模块：平面设计
- 命令名称：`RD_ALIGN_CURVE_PARAM_EDIT`
- 对应代码入口：`ObjectArxAlignmentCommand.cpp`、`AlignmentDialogBridge.*`、`HorizontalAlignmentBuilder.*`
- 业务文档维护人：RoadProto
- 原型版本：v0.1.8
- 是否可复用：是

## 功能背景

道路中线创建后需要调整半径和缓和曲线长度。该功能通过 WPF 参数页编辑每个交点对应的平曲线参数，并由 C++ 领域层统一重建道路中线。

## 业务目标

- 支持编辑每组平曲线的 `T1`、`T2`、`LS1`、`R`、`LS2`。
- 支持多交点控制点链按“第 x 个平曲线”翻页编辑。
- 保持五单元平曲线形态稳定，避免参数修改后出现折线穿插、反弯破坏或孤立短线。

## 适用场景

- 创建道路中线后调整圆曲线半径和缓和曲线长度。
- 多交点道路中线需要逐组校核平曲线参数。
- 后续夹点编辑后需要在参数窗口查看派生值。

## 输入条件

- CAD 选择对象：`DnRoadCenterlineEntity`。
- 用户输入参数：`LS1`、`R`、`LS2`；`T1/T2` 当前作为派生值展示和保护。
- 已有设计实体：道路中线实体及其控制点链。
- 外部数据：无。

## 输出结果

- CAD 图形实体：按新参数重建后的道路中线实体。
- 领域实体：更新后的 `HorizontalCurveParameters` 集合和 `HorizontalAlignment`。
- 表格或报告：无。
- 更新通知或重建请求：当前实体立即重建，后续联动模块暂不自动更新。

## 操作流程

1. 用户运行 `RD_ALIGN_CURVE_PARAM_EDIT` 或点击 Ribbon `编辑平曲线参数`。
2. 用户选择道路中线实体。
3. 系统打开 WPF 道路中线窗口中的平曲线参数页。
4. 用户按“第 x 个平曲线”翻页，编辑对应交点的参数。
5. WPF 将全部参数写入响应文件。
6. C++ 侧读取响应文件，按道路中线控制点链和参数集合重建实体。

## 关键业务规则

- `LS1` 与 `LS2` 可不等长。
- 领域层分别派生前切线长 `T1` 和后切线长 `T2`。
- `T1/T2` 当前作为派生值保护线形稳定，正式自由编辑模式后续再扩展。
- 无效半径、负缓和曲线长或切线空间不足时不得写入实体。
- WPF 只收集输入和展示结果，不直接计算平曲线主点。

## 可复用性说明

- 可复用内容：平曲线参数 DTO、五单元重建、参数有效性校验、多交点参数集合回写。
- 临时原型内容：WPF 请求/响应文件 Bridge。
- 正式复用前需要改造的内容：增加锁定半径、锁定缓和曲线、等比缩放、规范推荐参数等编辑控制模式。

## 与其他模块的依赖关系

- 上游模块：道路中线创建。
- 下游模块：夹点动态编辑、ICD 导出、纵断面和横断面。
- 实体联动行为：第一版仅重建当前道路中线实体。

## 后续对接正式 EICAD 功能的注意事项

- 平曲线参数规则必须保持在领域层。
- 后续开放 `T1/T2` 自由编辑前，需要先明确与半径、缓和曲线长度之间的约束关系。
