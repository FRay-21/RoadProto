# 竖曲线夹点与右键编辑

> 文档粒度要求：本文只描述竖曲线夹点、PVI 新增和 PVI 删除，不描述创建或 WPF 通用属性编辑。

## 基本信息

- 功能名称：竖曲线夹点与右键编辑
- 所属模块：纵断面设计
- 命令名称：`RD_PROFILE_VERTICAL_CURVE_ADD_PVI`、`RD_PROFILE_VERTICAL_CURVE_DELETE_PVI`
- 托管辅助命令：`RD_PROFILE_VERTICAL_CURVE_CONTEXT_ADD_PVI`、`RD_PROFILE_VERTICAL_CURVE_CONTEXT_DELETE_PVI`
- 对应代码入口：`DnProfileVerticalCurveEntity.*`、`ObjectArxProfileVerticalCurveCommand.*`、`RoadProtoRibbonExtension.cs`
- 业务文档维护人：RoadProto
- 原型版本：V0.1.9 follow-up
- 是否可复用：是

## 功能背景

竖曲线设计需要像折线夹点一样支持起点、终点和 PVI 拖动，同时每个 PVI 还需要通过半径夹点调整曲线半径。PVI 的新增和删除采用命令入口承接，并由托管 UI 接入 AutoCAD 对象右键菜单；用户选中竖曲线实体后右键，可进入新增或删除 PVI 流程。

## 业务目标

- 支持起点、终点和 PVI 夹点拖动，拖动后按拉坡图图面坐标反算桩号和设计高程。
- 支持半径夹点拖动，拖动后更新 PVI 半径并重算曲线长度、BVC/EVC 和采样点。
- 支持选择竖曲线后新增 PVI。
- 支持选择竖曲线后删除最近的 PVI。

## 输入条件

- CAD 对象：`DnProfileVerticalCurveEntity`。
- 用户输入：夹点拖动偏移，或在拉坡图上点取新增/删除 PVI 的位置。
- 已有数据：竖曲线起终点和可选 PVI 列表。

## 输出结果

- 更新起终点、PVI 或半径。
- 重新计算竖曲线元素和采样点。
- 刷新实体图形和几何范围。

## 夹点规则

夹点索引约定如下：

| 索引范围 | 含义 |
| --- | --- |
| `0` | 起点 |
| `1` | 终点 |
| `2..2+pviCount-1` | PVI 点 |
| `2+pviCount..2+pviCount*2-1` | 半径控制点 |

拖动起点、终点或 PVI 时，系统按关联拉坡图的局部坐标系反算桩号和高程。拖动半径夹点时，系统按半径夹点的新桩号偏移反算半径，因此半径可以增大也可以减小。

## 新增和删除 PVI

- `RD_PROFILE_VERTICAL_CURVE_ADD_PVI`：选择竖曲线实体，在拉坡图上点取新增位置，默认半径为 `1000`。
- `RD_PROFILE_VERTICAL_CURVE_DELETE_PVI`：选择竖曲线实体，在拉坡图上点取目标附近位置，删除桩号/高程距离最近的 PVI。
- 选中竖曲线实体后右键，AutoCAD 对象快捷菜单显示 `新增竖曲线变坡点` 和 `删除竖曲线变坡点`。
- `RD_PROFILE_VERTICAL_CURVE_CONTEXT_ADD_PVI` 和 `RD_PROFILE_VERTICAL_CURVE_CONTEXT_DELETE_PVI` 是托管 UI 菜单转发命令，分别调用上述 C++ 命令。

## 关键业务规则

- 起点和终点不可删除，但可以拖动。
- 竖曲线桩号范围可短于或长于关联拉坡图地面线范围。
- PVI 必须位于起终点之间，且曲线半径应能使 BVC/EVC 落在相邻坡段内。
- 无效拖动或无效 PVI 参数不会写回实体。
- 损坏数据、缺失起终点或非法图面映射时，实体不生成夹点，避免索引错位写坏数据。

## 后续扩展

- 右键菜单可进一步根据 PVI 数量动态禁用“删除竖曲线变坡点”。
- 后续可把右键点击位置直接作为新增/删除点位，减少二次点取。
- 增加夹点拖动过程中的连续动态预览。
- 在图面显示 BVC/EVC、高点或低点标记和坡度标注。
