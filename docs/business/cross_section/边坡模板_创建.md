# 边坡模板_创建

## 基本信息

- 功能名称：边坡模板创建
- 所属模块：横断面设计
- 命令名称：`RD_SECTION_SLOPE_TEMPLATE_CREATE`
- 对应代码入口：`src/cad_adapter/objectarx/cross_section/ObjectArxSlopeTemplateCommand.cpp`
- 业务文档维护人：RoadProto 项目
- 原型版本：`v0.1.14`
- 是否可复用：部分可复用

## 功能背景

边坡模板用于预先配置道路横断面放坡规则，后续在横断面戴帽中从路基模板最外侧向外搜索地面线，并生成道路边坡模型。当前版本先实现独立模板实体、WPF 参数窗口和线框预览。

## 业务目标

- 在图中点取插入位置，创建独立 `DnSlopeTemplateEntity`。
- 使用 WPF 窗口配置模板名称、显示比例、模板类型预设和部件参数。
- 支持填方、挖方和护坡道三类部件，并用线表达标准模板样式。
- 把边坡模板参数持久化到 DWG，供横断面戴帽引用。

## 操作流程

1. 运行 `RD_SECTION_SLOPE_TEMPLATE_CREATE` 或点击 Ribbon `横断面设计 / 创建边坡模板`。
2. 命令提示用户在图中点取边坡模板插入位置。
3. C++ 生成默认边坡模板数据，并通过桥接文件打开 WPF 参数窗口。
4. 用户在窗口中选择填方或挖方预设，继续调整部件、参数和控制条件。
5. 用户确认后，WPF 写回响应文件。
6. C++ 读取响应文件，创建 `DnSlopeTemplateEntity` 并刷新图形。

## 默认参数

- 默认名称：`边坡模板1`
- 显示比例：`1:10`
- 默认部件：边坡、护坡道、边坡、护坡道、边坡
- 填方边坡：坡率 `1:1.5`，坡高 `4`，搜索地面线范围坡高增值 `2`
- 挖方边坡：坡率 `1:-1.5`，坡高 `4`，搜索地面线范围坡高增值 `2`
- 护坡道：宽度 `1`，横坡 `3%`

## 关键业务规则

- 填方模板和挖方模板只是初始参数预设，用户可以在同一模板内自由调整部件类型和参数。
- 每个部件支持三种编辑方式：坡率+坡高、坡率+宽度、坡高+宽度；第三个值由规则计算。
- 护坡道同样支持三种编辑方式，默认使用宽度与横坡。
- `搜索地面线范围坡高增值` 是每个部件自己的配置值；为空或为 `0` 时不生效。
- `交地则结束放坡` 关闭时，模板按配置正常生成，不要求必须交地才算有效。
- `重复最后一组参数直至交地` 的最后一组定义为最后一个护坡道加最后一个边坡。
- CAD 实体预览只显示线，不表达边坡厚度、面、材质或算量。

## 当前限制

- 第一版本只生成模板线框实体，不生成边坡实体面。
- 预览只表达标准坡率、宽度和坡高，不预览更复杂的交地条件。
- 模板库、批量套用、正式关系联动和自动重建仍在后续版本扩展。
