# 竖曲线夹点与右键编辑

> 文档粒度要求：本文只描述竖曲线 PVI 夹点和右键编辑入口，不描述创建或通用属性编辑。

## 基本信息

- 功能名称：竖曲线夹点与右键编辑
- 所属模块：纵断面设计
- 命令名称：`RD_PROFILE_VERTICAL_CURVE_ADD_PVI`、`RD_PROFILE_VERTICAL_CURVE_DELETE_PVI`
- 对应代码入口：`src/modules/profile/ProfileModule.cpp`
- 业务文档维护人：RoadProto
- 原型版本：V0.1 ProfileGradeGraph
- 是否可复用：部分可复用

## 功能背景

竖曲线设计需要支持对变坡点 PVI 的增删和局部调整。当前 Task 4 只登记命令元数据和业务文档，后续任务再实现右键菜单、夹点动态预览和实体更新。

## 业务目标

提供新增 PVI 和删除 PVI 的原型命令入口，为后续竖曲线夹点编辑、右键菜单和参数化重建流程预留稳定命令名。

## 适用场景

适用于已经存在竖曲线实体的纵断面图，用于后续交互式增加或删除竖曲线变坡点。

## 输入条件

- CAD 选择对象：后续选择竖曲线实体或其 PVI 夹点。
- 用户输入参数：后续输入 PVI 桩号、高程或从鼠标点位推导。
- 已有设计实体：竖曲线实体。
- 外部数据：无。

## 输出结果

- CAD 图形实体：后续更新竖曲线实体和 PVI 标记。
- 领域实体：后续更新 PVI 列表和竖曲线计算结果。
- 表格或报告：无。
- 更新通知或重建请求：后续触发关联显示重建。

## 操作流程

1. 用户通过后续右键菜单或命令调用新增/删除 PVI。
2. C++ CAD adapter 负责定位实体、点位和 handle。
3. application/domain 层校验 PVI 顺序、参数合法性并计算新曲线。
4. cad_adapter 将结果写回 DWG 实体并刷新显示。

## 关键业务规则

- `RD_PROFILE_VERTICAL_CURVE_ADD_PVI` 用于新增竖曲线变坡点。
- `RD_PROFILE_VERTICAL_CURVE_DELETE_PVI` 用于删除竖曲线变坡点。
- WPF 只负责必要的 DTO/参数输入和提示。
- 核心计算、PVI 列表修改、实体写回和 CAD 刷新由 C++ application/domain/cad_adapter 执行。
- 当前阶段不实现真实右键菜单和夹点动态交互。

## 可复用性说明

- 可复用内容：PVI 增删命令边界、后续 PVI 参数校验和重建服务。
- 临时原型内容：当前命令过程为空实现，只用于框架接线。
- 正式复用前需要改造的内容：补充夹点命中、右键入口、事务封装、动态预览和错误恢复。

## 与其他模块的依赖关系

- 上游模块：竖曲线创建、竖曲线编辑。
- 下游模块：纵断面显示、出图和算量。
- 实体联动行为：PVI 增删后应通过统一关系管理机制通知依赖对象。

## 后续对接正式 EICAD 功能的注意事项

- 保持 PVI 编辑服务独立于 UI 技术栈。
- 夹点动态预览应由 ObjectARX adapter 负责 CAD 交互，由 domain/application 负责参数和计算规则。
