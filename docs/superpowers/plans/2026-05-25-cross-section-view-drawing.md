# 查看横断面批量落图实施计划

> 本文件为历史实施计划的中文归档版。以后 `docs/superpowers/plans/` 下的计划文档统一使用中文编写；代码标识、命令名、路径和构建命令保持原文。

## 目标

在查看横断面窗口中增加“绘制横断面”，批量生成 `DnRoadModelSectionDrawingEntity` 横断面图实体，为后续配置、标注和算量提供图上承载对象。

## 主要任务

- 扩展道路模型查看 Bridge，支持 WPF 响应绘制动作。
- 新增横断面图实体数据结构、显示、DWG 持久化和几何范围。
- 在 ObjectARX 中按桩号批量落图，并使用白色外框和桩号文字。
- 将结构层面域、颜色和填充参数写入横断面图实体。
- 更新业务文档、模块文档、复用说明和测试说明。

## 验证

- 核心测试覆盖横断面落图实体源码契约和绘制流程。
- AutoCAD 手工验证点取基点、批量落图、结构层面域、保存重开和显示效果。
