# 纵断面拉坡图创建

## 功能范围

本功能属于纵断面设计模块，当前为增量原型实现。Task 3 仅注册命令元数据和 C++ Ribbon model 面板，不创建实体、不执行 CAD 选择/采样、不打开 WPF 窗口。

## 命令

- 用户命令：`RD_PROFILE_GRADE_GRAPH_CREATE`
- 显示名称：纵断面拉坡图
- 模块编码：`PROFILE`
- Ribbon 位置：`RoadProto / 纵断面设计 / 纵断面拉坡图`

## 后续实现

后续任务将补充纵断面拉坡图实体、CAD 数据选择/采样、WPF 参数窗口和可见 AutoCAD Ribbon 扩展。
