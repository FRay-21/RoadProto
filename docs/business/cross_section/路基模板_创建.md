# 路基模板创建

## 功能范围

本功能属于横断面设计模块，用于创建独立的路基模板自定义实体。当前版本只要求用户点取插入点，不选择、不绑定既有道路中线；实体自身绘制道路中线标记，并预留后续与道路中线、路面结构层关联的字段。

## 命令

- 用户命令：`RD_SECTION_SUBGRADE_TEMPLATE_CREATE`
- 显示名称：创建路基模板
- 模块编码：`CROSS_SECTION`
- Ribbon 位置：`RoadProto / 横断面设计 / 创建路基模板`
- 自定义实体：`DnSubgradeTemplateEntity`
- DXF 名称：`DNSUBGRADETEMPLATEENTITY`

## 流程

1. 用户运行 `RD_SECTION_SUBGRADE_TEMPLATE_CREATE`。
2. 命令提示用户在图中点取路基模板实体插入位置。
3. C++ 生成默认模板数据，并通过桥接文件打开 WPF 参数窗口。
4. 用户在窗口中编辑模板名称、显示比例、道路等级和部件参数。
5. 用户确认后，WPF 写回响应文件。
6. C++ 读取响应文件，创建 `DnSubgradeTemplateEntity` 并刷新图形。

## 初始参数

高速公路默认部件：

- 左侧中分带 `1.5`
- 左侧路缘带 `0.75`
- 左侧行车道 `3.75 * 2`
- 左侧硬路肩 `3`
- 左侧土路肩 `0.75`
- 右侧对称

城市快速路默认部件：

- 左侧中分带 `1`
- 左侧行车道 `3.75 * 2`
- 左侧侧分带 `1`
- 左侧慢车道 `3`
- 左侧人行道 `4`
- 右侧对称

二级、三级、四级道路默认部件：

- 左侧行车道 `3.75`
- 左侧行车道 `3.75`
- 左侧硬路肩 `3`
- 左侧土路肩 `0.75`
- 右侧对称

一级道路、城市主干道、城市次干道和城市支路本版先生成空模板，后续按业务规则扩展。

## 部件规则

- 左侧部件只在中线左侧绘制，右侧部件只在中线右侧绘制。
- 宽度变化表按“桩号、加宽值”保存；当前未关联路线桩号，实体图形和预览先按基础宽度显示。
- 后续绑定道路中线后，实际宽度按 `基础宽度 + 加宽值` 计算，且向中线外侧加宽。
- 坡度支持固定值和变化值；选择变化值后，固定坡度输入置灰，坡度只能从“坡度变化数据表”获取。
- 当前未关联路线桩号，实体图形和预览在变化坡度模式下先使用坡度变化表首行作为示意坡度；后续绑定道路中线后按桩号查询坡度变化表。
- 路面结构层关联字段本版只预留 UI 和数据接口，启用关联后才保留厚度。
- 二维预览图中每个部件应显示部件类型、宽度和坡度，并裁剪在预览区域内，不得遮挡右侧参数 UI。
- CAD 图中的路基模板自定义实体部件标注使用中文部件名。

## 颜色约定

- 中分带、侧分带：RGB `0,120,0`
- 行车道、硬路肩、慢车道、路缘带：RGB `0,90,180`
- 土路肩、人行道：RGB `120,120,120`

## 当前限制

- 本版不选择、不绑定道路中线。
- 变宽表和坡度变化数据表暂不按桩号实时驱动图形。
- 路面结构层实体尚未实现，仅预留句柄和厚度字段。
- AutoCAD 图形界面中的真实 Ribbon 点击、双击编辑、DWG 保存重开仍需人工验证。
## 2026-05-13 更新

- 空白新建请求必须按高速公路默认值打开；已有实体编辑请求保留实体保存的道路等级和部件列表。
- 一级道路、二级道路、三级道路、四级道路、城市主干道、城市次干道和城市支路均提供非空初始部件配置。
- 预览图部件通过透明命中区域支持直接点选；左、右按钮按横断面几何顺序移动，不再按原始列表顺序绕行。
