# 路面结构层模板通用参数设计

## 背景

路面结构层模板现有通用参数只包含模板名称、显示比例、预览宽度、结构层数量、显示方式和当前编辑层。用户需要先把规范类属性随模板保存，暂不参与预览图、DWG 模板实体或道路模型显示。

## 目标

- 在路面结构层模板属性中新增结构代号、路基干湿类型、路面类型、路基土组、设计弯沉和累计轴次。
- 新增“显示全部通用参数”开关，默认关闭；关闭时右侧通用参数只显示现有四项：预览宽度、结构层数量、显示方式、当前编辑层。
- 打开开关后显示新增的全部通用参数。
- 新增参数通过 WPF 编辑，并随 Bridge 请求/响应、`.rpavement.xml` 和 DWG 自定义实体持久化。
- 新增参数仅作为数据保留，不影响 WPF 预览图、不显示到 DWG 模板实体、不改变道路模型结构层颜色显示。

## 数据模型

`PavementLayerTemplateProperties` 增加：

- `structureCode`：结构代号，文本，默认空。
- `subgradeMoistureTypes`：路基干湿类型，多选，稳定编码保存。
- `pavementType`：路面类型，单选，默认沥青路面。
- `subgradeSoilGroups`：路基土组，多选，稳定编码保存。
- `designDeflection`：设计弯沉，文本，默认空。
- `cumulativeAxleLoads`：累计轴次，文本，默认空。
- `showAllGeneralParameters`：是否显示全部通用参数，默认 `false`。

多选字段使用分号分隔编码在 Bridge/XML/DWG 中传递，UI 显示为中文复选框列表。文本字段不参与数值校验，避免当前原型阶段误限制用户输入单位或表达格式。

## UI

右侧“通用参数”区域保持现有四项常驻。其下新增“显示全部通用参数”复选框，默认不勾选。勾选时展开高级通用参数区：

- 结构代号：文本框。
- 路基干湿类型：干燥、中湿、潮湿、过湿，多选。
- 路面类型：沥青路面、混凝土路面，下拉单选。
- 路基土组：基岩、碎石土、砾类土、砂类土、粉质土、低液限黏土、高液限黏土、有机质土、软土、膨胀土、黄土、其他，多选。
- 设计弯沉：文本框。
- 累计轴次：文本框。

## 持久化

Bridge 请求文件、响应文件、XML 和 DWG 都保存新增字段。旧 DWG 版本读取时使用默认值；新版本实体版本号递增。

## 测试

- 核心测试先检查领域默认值、枚举编码、归一化、版本和 ObjectARX 源码契约。
- 托管 Bridge 测试先检查请求/响应、XML 往返、WPF 控件契约和默认隐藏开关。
- 构建验证使用 Release 全量 solution、核心测试和托管 bridge 测试。
