# 路面结构层模板_编辑

## 功能边界

`RD_SECTION_PAVEMENT_LAYER_TEMPLATE_EDIT_HANDLE` 用于按 handle 打开既有 `DnPavementLayerTemplateEntity` 的 WPF 编辑窗口。

## 当前流程

1. 命令接收或读取模板实体 handle。
2. ObjectARX 层解析 handle 并校验对象类型。
3. Bridge 将实体保存的插入点和模板数据写入 WPF 请求文件。
4. WPF 编辑完成后通过回写命令更新原实体。

## 分层约束

- 编辑命令只做 CAD 对象定位、类型校验和桥接调用。
- 模板数据校验与归一化继续复用领域层 `PavementLayerTemplateRules`。
