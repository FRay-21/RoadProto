# 道路模型生成进度与边坡模板组 UI 实施计划

> 本文件为历史实施计划的中文归档版。以后 `docs/superpowers/plans/` 下的计划文档统一使用中文编写；代码标识、命令名、路径和构建命令保持原文。

## 目标

为横断面戴帽增加生成进度反馈，并完善左右侧边坡模板组管理交互。

## 主要任务

- 在道路模型构建输入中增加进度回调。
- 在 ObjectARX 命令中接入 AutoCAD 状态栏进度条。
- 在 WPF 中完善模板组选择、点选、删除、上移、下移和清空。
- 更新 Bridge、测试和业务文档。

## 验证

- 核心测试覆盖进度回调和模板组优先级。
- AutoCAD 手工验证生成进度、模板组管理和模型生成结果。
