# 道路模型三维网格线框实施计划

> 本文件为历史实施计划的中文归档版。以后 `docs/superpowers/plans/` 下的计划文档统一使用中文编写；代码标识、命令名、路径和构建命令保持原文。

## 目标

增强 `DnRoadModelEntity` 的三维网格表达，绘制横向肋线、纵向连接线、最外侧边界线、端部封闭线和节点数量不一致时的过渡线。

## 主要任务

- 在道路模型领域数据中生成可持久化的网格线框。
- 在实体绘制中增加网格线显示和 DWG 保存读取。
- 补充核心测试、源码契约、复用说明和版本记录。

## 验证

- 核心测试覆盖线框生成和过渡线。
- AutoCAD 手工验证道路模型显示、保存重开和 `REGEN`。
