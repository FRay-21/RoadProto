# 道路模型_WPF桥接回写

## 命令

- `RD_SECTION_ROAD_MODEL_APPLY_DIALOG_FILE`

## 范围

道路模型 WPF 编辑结果回写的内部桥接命令说明。该命令面向 WPF 与 C++ ObjectARX 之间的文件桥接回写流程，不作为 Ribbon 用户按钮。

## 流程

1. WPF 后续编辑窗口将道路模型编辑结果写入约定的临时文件。
2. 桥接命令读取文件并转发给 C++ 应用层处理。
3. 后续任务再实现数据解析、校验和实体刷新。

## 当前限制

- 当前仅建立内部命令文档和路径契约。
- 不定义 DTO 字段，不实现对话框文件解析、CAD 实体更新或错误处理细节。
