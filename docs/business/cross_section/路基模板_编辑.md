# 路基模板编辑

## 功能范围

本功能用于编辑已创建的 `DnSubgradeTemplateEntity`。用户可通过双击实体触发托管 Ribbon 插件转发，也可由内部句柄命令打开同一个 WPF 参数窗口。

## 命令

- 内部命令：`RD_SECTION_SUBGRADE_TEMPLATE_EDIT_HANDLE`
- 显示名称：按句柄编辑路基模板
- 模块编码：`CROSS_SECTION`
- 业务文档：`docs/business/cross_section/路基模板_编辑.md`

## 流程

1. 托管 Ribbon 插件监听 AutoCAD 双击事件。
2. 若双击对象 DXF 名称为 `DNSUBGRADETEMPLATEENTITY`，插件读取对象句柄。
3. 插件转发命令：`RD_SECTION_SUBGRADE_TEMPLATE_EDIT_HANDLE <handle>`。
4. C++ 按句柄打开实体，读取插入点和模板数据。
5. C++ 通过桥接文件打开 WPF 参数窗口。
6. WPF 参数窗口必须显示实体内上次保存的模板名称、道路等级、显示比例、部件列表、坡度模式、变宽表、坡度变化表和路面结构层预留字段，不得重新初始化为默认模板。
7. 用户确认后，C++ 读取响应文件并回写实体。

## 可编辑内容

- 模板名称。
- 显示比例。
- 道路等级。
- 左右侧部件列表。
- 部件类型、宽度、高度差、坡度、颜色。
- 变宽表、坡度变化数据表。
- 路面结构层关联预留字段和厚度。
- CAD 图中部件标注使用中文部件名；WPF 预览图显示部件宽度和坡度。

## 边界

- WPF 不直接操作 `AcDbEntity`、`AcDbObjectId`、`ads_name`。
- C++ 回写前会做基础参数归一化与校验。
- 当前未实现按路线桩号重算变宽/变坡图形。
## 2026-05-13 更新

- 双击编辑必须发送带换行的 `RD_SECTION_SUBGRADE_TEMPLATE_EDIT_HANDLE <handle>`，确保句柄参数立即被命令读取。
- WPF 编辑模式必须从实体读取已保存的部件、道路等级、数据表和预览状态，不得重新生成默认模板。
- 预览图支持通过部件本体和命中区域直接选中部件；左、右按钮按横断面几何顺序移动。
