# 路面结构层模板_创建

## 功能边界

`RD_SECTION_PAVEMENT_LAYER_TEMPLATE_CREATE` 用于在 CAD 图中创建独立的路面结构层模板实体。

## 当前流程

1. 用户执行命令后点取模板插入位置。
2. C++ 应用服务生成默认路面结构层模板数据。
3. ObjectARX 命令立即创建 `DnPavementLayerTemplateEntity`，写入插入点和默认模板数据，并追加到模型空间。
4. 命令读取新实体 handle，将该 handle、插入点和模板数据写入 WPF 请求文件，并打开 WPF 编辑窗口。
5. 用户确认后由桥接回写命令按 handle 更新既有 `DnPavementLayerTemplateEntity`。

## 分层约束

- 默认数据生成位于 `application/cross_section`。
- 路面结构层规则位于 `domain/cross_section`。
- CAD 实体创建、持久化和图面预览位于 `cad_adapter/objectarx`。
- WPF 只负责参数编辑和详细预览。
