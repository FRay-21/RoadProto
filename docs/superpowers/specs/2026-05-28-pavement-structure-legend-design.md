# 路面结构图例设计

## 背景

出图出表模块已有路面工程量统计表，能够从横断面图实体读取当前结构层面域。新增“路面结构图例”用于在图纸中生成普通 CAD 图形，表达当前道路实际使用到的路面结构层模板、模板设计参数、层厚和填充含义。

## 用户入口

- 新增命令：`RD_DRAWING_PAVEMENT_STRUCTURE_LEGEND`
- Ribbon 位置：`RoadProto / 出图出表 / 路面结构图例`
- 用户运行命令后，提示选择道路模型或横断面图。
- 如果选择道路模型，则以该 `DnRoadModelEntity` 为来源。
- 如果选择一个横断面图，则读取其 `roadModelHandle`，扫描当前模型空间中同一道路模型的所有 `DnRoadModelSectionDrawingEntity`。
- 收集完成后提示用户点取图例插入位置。

## 数据来源

优先从同一道路模型的横断面图结构层面域读取 `sourceTemplateHandle`，因为横断面图配置后的结构层面域代表出图状态。若图中没有可用模板 handle，则兼容读取道路模型 `pavementLayerLines` 中引用的路面结构层模板 handle。

每个模板 handle 解析为 `DnPavementLayerTemplateEntity`，读取 `PavementLayerTemplateData`：

- `properties.structureCode`
- `properties.subgradeSoilGroups`
- `properties.subgradeMoistureTypes`
- `properties.designDeflection`
- `properties.cumulativeAxleLoads`
- `layers[].name`
- `layers[].type`
- `layers[].thickness`、`innerThickness`、`outerThickness`
- `layers[].hatchPattern`、`hatchAngle`、`hatchScale`
- `layers[].color`

模板列按首次在横断面图或道路模型中出现的顺序去重。模板本身去重，底部填充图例不合并。

## 图面布局

生成内容全部是普通 CAD 实体，不创建新的自定义实体。默认使用黄色文字、白色表格线和白色/模板样式填充示意。

图例表按统一列宽绘制。左侧为行标题，右侧每个路面结构层模板一列。行顺序为：

1. 路基土组
2. 路基干湿类型
3. 设计弯沉
4. 累计当量轴次
5. 结构代号
6. 结构图示
7. 路面总厚度(cm)
8. 图例

结构图示行中，每列使用固定示意宽度 `20cm`，按模板结构层自上而下绘制。厚度按厘米 1:1 转换，模板层厚以米保存时换算为厘米。若内外侧厚度不一致，图示高度使用内外侧厚度的平均值，厚度文字显示为 `内/外` 两个值；若一致则显示单值。每列内容在该列水平居中。

总厚度行显示各层厚度合计，非等厚层使用内外平均厚度参与合计，并在结构图示旁保留每层厚度文字。

## 底部图例

底部“图例”行按模板层逐项生成填充样式说明。即使多个结构层使用相同填充样式，只要结构层名称不同，也分别绘制独立图例项；不按填充样式合并。

图例项内容包括：

- 一个固定尺寸的填充示例矩形。
- 对应结构层名称，优先使用 `layer.name`，为空时使用结构层类型中文显示名。

图例项按模板列顺序、层顺序排列，必要时在底部行内自动横向排布并扩展总宽度。

## 领域与 CAD 边界

- `domain/quantity` 新增路面结构图例绘制计划模型，负责模板字段格式化、厚度换算、列宽和图例项数据整理，不依赖 ObjectARX。
- `cad_adapter/objectarx/drawing_quantity` 负责选择对象、读取实体、点取插入点、创建普通 CAD 实体和追加到模型空间。
- `modules/drawing_quantity` 只注册命令和 Ribbon 元数据。
- WPF 托管 Ribbon 只新增按钮并发送命令，不承载业务逻辑。

## 错误处理

- 未选择道路模型或横断面图：提示并退出。
- 横断面图缺少道路模型 handle：提示并退出。
- 当前道路未找到任何路面结构层模板：提示并退出。
- 某个模板 handle 无法解析：跳过该模板并在命令行输出警告；若最终没有有效模板则退出。
- 用户取消插入点：提示取消，不绘制任何实体。

## 测试

核心测试覆盖：

- 路面结构图例命令元数据和 Ribbon 入口。
- 绘制计划按模板顺序去重。
- 结构代号、路基土组、路基干湿类型、设计弯沉和累计当量轴次从模板属性格式化。
- 层厚从米换算为厘米，等厚显示单值，非等厚显示内/外值，总厚度使用平均厚度。
- 底部图例按模板层逐项输出，同填充样式不同层名不合并。
- ObjectARX 命令源码契约包含道路模型/横断面图双入口、模型空间扫描、普通实体绘制和插入点点取。

AutoCAD 手工验证覆盖：

- 选择道路模型生成图例。
- 点选一个横断面图后自动搜索同一道路模型全部横断面图。
- 生成内容均为普通 CAD 对象，可单独选择、移动、删除。
- 结构图示列宽一致，内容居中，厚度按厘米 1:1 表达，宽度固定为 `20cm`。
- 底部图例中同填充样式但不同结构层名称分别显示。
