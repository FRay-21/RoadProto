# 路面工程量统计表部件拆分模式实施计划

> 本文件为历史实施计划的中文归档版。以后 `docs/superpowers/plans/` 下的计划文档统一使用中文编写；代码标识、命令名、路径和构建命令保持原文。

## 目标

为 `RD_DRAWING_PAVEMENT_QUANTITY_TABLE` 增加统计方式选择，支持按“部件名称-结构层名称”拆列，也保留按结构层类型汇总模式。

## 主要任务

- 扩展领域层统计模型，增加 `PavementQuantityAggregationMode`。
- 横断面图面域和道路模型采样保留部件名称。
- 保存文件对话框增加统计方式单选项。
- 更新命令元数据测试、业务文档、模块文档、复用说明和测试说明。

## 验证

- 核心测试覆盖两种聚合模式、动态列名、面积和体积。
- AutoCAD 手工验证保存对话框选项和 `.xls` 输出列。
